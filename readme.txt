Video Share / Video on Demand (VOD) plugin allows WordPress users and admins to share videos and others to watch from various devices. Optionally, integrates with VideoWhisper Live Streaming. 


== Description ==

= Key Features =
* adds video post type to WordPress site
* extracts thumbnail, generates feature image
* extracts info: duration, resolution, bitrate, file size
* HTML5 video conversion and playback support
* RTMP playback support
* HLS playback support
* Import and show archived video streams for VideoWhisper Live Streaming channels plugin
* AJAX display and update of video list
* shortcodes for listing videos, displaying player

= Special Requirements =
This plugin has requirements beyond regular WordPress hosting specifications:
* FFMPEG and codecs are required to generate snapshots
* For RTMP playback, RTMP hosting is required. For HLS playback, a server with HLS support like Wowza is required.
